## EditColorHelper

更改 EditText 的各种颜色

### 如何使用？
1.在你项目的 build.gradle 文件中添加
```
allprojects {
    repositories {
    ...
    maven { url 'https://jitpack.io' }
    }
}
```
2.配置依赖
```
dependencies {
    compile 'com.github.iAcn:EditColorHelper:v1.0'
}
```
3.尽情使用吧
```
EditHelper.setColor(EditText, Color);
```
